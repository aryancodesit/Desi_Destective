## üíª Installation & Setup

Want to play locally or contribute? Follow these steps:

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn

### Steps
1.  **Clone the Repository**
    ```bash
    git clone https://github.com/aryancodesit/Desi_Destective.git
    cd Desi_Destective
    ```

2.  **Install Dependencies**
    ```bash
    npm install
    # or
    yarn install
    ```

3.  **Run the OS**
    ```bash
    npm run dev
    ```
    Open [http://localhost:3000](http://localhost:3000) in your browser to enter to your detective PC.
